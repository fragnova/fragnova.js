Function getGenealogy

  • getGenealogy(getProtosGenealogyParams: getProtosGenealogyParams): Promise<any>

  • Return the Proto Genealogy, get the descendants and ancestors of a Proto. The return type is a JSON string that represents an Adjacency List.

    Example
